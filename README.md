# WebShare — P2P File Transfer

A premium, peer-to-peer file transfer web application built with **WebRTC** and **Vite**.

## Features

- **Direct P2P**: Files flow directly between browsers using PeerJS.
- **Privacy First**: No server-side storage; your data stays in your browser.
- **Easy Link**: Share a 6-character room code to connect devices.
- **Modern UI**: Dark-themed glassmorphism design with responsive support.

## Installation & Running Locally

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```

## Deployment

This project is ready to be deployed to **GitHub Pages**.

1. Run the deploy script:
   ```bash
   npm run deploy
   ```
2. Your app will be accessible at `https://<your-username>.github.io/<repo-name>/`.
